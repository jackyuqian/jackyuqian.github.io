# {{ page.title }}

## Modify ~/.vimrc
~~~bash
" Color Scheme"
colorscheme desert

" Syntax Highlight"
filetype on
filetype plugin on
filetype indent on
syntax enable

" No Backup"
set nobackup

" Indent"
set autoindent
set cindent
set shiftwidth=4

" Tab Stop"
set tabstop=4
set softtabstop=4
set expandtab

" Line Number"
set nu

" Search"
set hlsearch
set incsearch

" Mouse Enable"
set mouse=a

" Bottom Information"
set ruler
~~~
